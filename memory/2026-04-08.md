## 便笺项目 Sticky Notes

用户想要一个类似微软便笺的 PC 端便笺软件，自己用。

### 过程
1. 最初尝试 Electron 方案，但服务器环境无显示器且 npm install 太慢，放弃
2. 改为单文件 HTML 方案（`index.html`），浏览器直接打开，localStorage 存储
3. 经过多轮迭代优化：
   - 第一版：基础拖拽、调整大小、6 色、自动保存
   - 用户反馈"排列、颜色、字体"需要优化 → 全量升级：8 色、自动排列、搜索、时间戳、右键菜单、导出、置顶、折叠、复制
   - 加网格背景线（20px 对齐）
   - 用户反馈菜单栏不好看、颜色不好看 → 重做整个 CSS：毛玻璃工具栏、胶囊按钮、8 种鲜艳配色（amber/rose/azure/jade/iris/graphite/tangerine/teal）
   - 去掉拖动旋转效果（用户觉得没必要）
   - 折叠/置顶按钮图标动态切换
   - 双击空白新建、Esc 优先清搜索、拖拽边界限制、hover 边框发光
   - 加浅色模式（真实便利贴风格）
   - 加快捷键面板（按 ? 弹出）
   - 修复搜索+排列 bug（隐藏便笺跳过）

### 功能清单
- 拖拽移动 & 调整大小（网格吸附 20px）
- 8 种鲜明配色 + 深色/浅色主题
- 搜索、自动排列、置顶、折叠、复制
- 右键菜单、时间戳、导出 JSON
- 快捷键：Ctrl+N 新建、Ctrl+B/I/U 格式化、? 帮助
- localStorage 自动保存
