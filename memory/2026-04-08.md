# 2026-04-08

## 墨滴 Modī 项目（原 Sticky Notes）

用户想要一个类似微软便笺的 PC 端便笺软件，自己用。

### 上午：基础开发
1. 最初尝试 Electron 方案，但服务器环境无显示器且 npm install 太慢，放弃
2. 改为单文件 HTML 方案（`index.html`），浏览器直接打开，localStorage 存储
3. 经过多轮迭代优化：
   - 第一版：基础拖拽、调整大小、6 色、自动保存
   - 用户反馈"排列、颜色、字体"需要优化 → 全量升级：8 色、自动排列、搜索、时间戳、右键菜单、导出、置顶、折叠、复制
   - 加网格背景线（20px 对齐）
   - 重做整个 CSS：毛玻璃工具栏、胶囊按钮、8 种鲜艳配色
   - 去掉拖动旋转效果、折叠/置顶按钮图标动态切换
   - 双击空白新建、Esc 优先清搜索、拖拽边界限制、hover 边框发光
   - 加浅色模式、快捷键面板（按 ? 弹出）
   - 修复搜索+排列 bug（隐藏便笺跳过）

### 下午：功能增强 + 重命名
- 重命名为「墨滴 Modī」，新设计 logo（紫色双层便笺 + 墨滴点缀）
- 新增暖心消息（8 时段 × 10 句，点击切换随机话语）
- 新增单实例锁定
- 工具栏瘦身（56px→40px）
- 新增 Checklist 模式、字体大小调节、图片粘贴支持

### 下午（续）：CI/CD 搭建
- 写了 GitHub Actions workflow（build.yml），但首次 push 因 token 缺 workflow scope 失败
- 用户提供新 token，成功推上 build.yml
- 首次 CI 打包失败：electron-builder 自动发布报 403 Forbidden
  - 原因：GITHUB_TOKEN 在 Actions 中权限不足创建 Release
  - 修复：加 `--publish never` + `permissions: contents: write`，Release 交给 softprops/action-gh-release
- v1.0.0 CI 构建成功，自动生成 Modi.Setup.1.0.0.exe

### 下午（续）：UI 优化（5 项改进）
1. **导入 JSON**：工具栏新增导入按钮，支持导入之前导出的便笺备份
2. **搜索高亮**：匹配关键词用黄色背景高亮显示
3. **品牌 logo 左边距**：加 4px margin-left
4. **Markdown 快捷**：输入 `- `、`* `、`# `、`> ` 后 Enter 自动格式化
5. **图片预览**：点击图片改为弹出浮层预览（带关闭按钮、Esc 关闭），解决容器内放大溢出问题

### 当前功能清单
- 拖拽移动 & 调整大小（网格吸附 20px）
- 8 种鲜明配色 + 深色/浅色主题
- 搜索（高亮匹配）、自动排列（Masonry）、置顶、折叠、复制
- 右键菜单、时间戳、导出/导入 JSON
- Checklist 模式、字体大小调节、图片粘贴 + 预览
- 快捷键：Ctrl+N 新建、Ctrl+B/I/U 格式化、Ctrl+Z/Y 撤销重做、? 帮助
- 暖心消息（8 时段自动切换，点击随机）
- localStorage 自动保存、单实例锁定
